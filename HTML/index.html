The Core Formula
The GPA calculation is a weighted average — a concept from Engineering Mathematics II.
% MATLAB — one line to calculate GPA
GPA = sum(credits .* gradePoints) / sum(credits);

% Example — Aarav Sharma, Semester 1
% Credits: [4, 4, 3, 2, 4, 3 ] → total = 20
% GradePoints: [9, 8, 9, 10, 10, 8 ]
% Weighted: [36, 32, 27, 20, 40, 24] → sum = 179
GPA = 179 / 20 = 8.95 → Very Good
